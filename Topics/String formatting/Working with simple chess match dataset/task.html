Working with simple chess match dataset

Imagine we design a machine that stores and loads chess results between 2 competitors. Each battle has 3 matches. The result is the color of the winner. For example, White White Black means that the white side has won the first and the second match and the black side has won the third match.

Now we have a dataset that stores the result of 5 battles. Can you implement the additional code required to print this dataset in the same format as below?

Note that the new line \n and tab \t characters will not be interpreted within backticks. For an easier time solving this task, you should not change the indents in the code snippet of the task prepared for you.

White White Black
Black Black White
White Black White
Black White Black
Black Black Black


Memory limit: 256 MB
Time limit: 8 seconds
